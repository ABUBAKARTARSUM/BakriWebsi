Pengenalan Apa itu HTML ?

HTML adalah singkatan dari Hypertext Markup Language merupakan bahasa yang digunakan untuk membuat struktur website, ibaratnya seperti ini jika kalian ingin membuat suatu rumah yang pertama di lakukan adalah membuat pondasi atau bangunannya kan? Nah, HTML juga demikin fungsi nya adalah membuat pondasi atau struktur dari Website.

format penulisan HTML

untuk membuat suatu HTML standar digunakan tag html, head, dan body beserta tag pembuka dan penutupnya masing - masing , susunan tag - tag tersebut mmebentuk dokumen HTML.

memberikan informasi dokumen HTML pada head

Apa fungsi head dalam HTML ? Tag head tugasnya adalah memberikan informasi tentang dokumen, maksudnya didalam tag head kita bisa menambahkan tag- tag yang biasanya digunakan untuk memberikan informasi berupa penulis judul dokumen contohnya tag title, kata kunci pada dokumen, tag link untuk menghubungkan ke file lain dan masih banyak lagi informasi yang bisa di tambahkan pada tag ini.

apa fungsi dari tag meta dalam HTML ? Elemen meta adalah elemen atau tag dalam bahasa pemrograman HTML atau XHTML yang berfungsi sebagai pemberi informasi metadata dari sebuah halaman web. Elemen meta ini harus diletakkan pada bagian “head” dalam HTML tersebut.

apa fungsi dari atribute "charset" ? Atribut charset pada meta tag digunakan untuk memberi tahu web browser dengan karakter set apa halaman web akan ditampilkan. Biasanya atribut charset ini akan berisi nilai utf-8, utf-8 adalah singkatan dari Uniform Transform Code-8 nilai ini merupakan nilai untuk HTML 5 / versi terbaru.

Apa yang di maksud dari tag title dan apa fungsinya ? Title tag adalah tag untuk membuat sebuah judul dari sebuah halaman.Lalu apa fungsinya, nah fungsinya adalah sebagai backlink yang akan menghubungkan user kepada situs anda. Title tag pada tampilan mesin pencari akan menarik user untuk mengklik website anda. Saat user melakukan kunjungan ke website anda, itulah yang disebut dengan visit.
